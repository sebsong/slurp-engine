# Slurp Engine
A souped up C++ game engine

## Windows Setup

1. Run `slurp-engine/SlurpEngine/commands/shell.bat`
2. Run `slurp-engine/SlurpEngine/commands/build.bat`
3. Run `slurp-engine/SlurpEngine/commands/run.bat`

### Notes

- It's useful to setup command prompt to run `shell.bat` on startup:

    ```cmd.exe /k slurp-engine\SlurpEngine\commands\shell.bat```
 

- Once `shell.bat` is run, the following commands can be run natively:
 
  - `build`
  - `run`
  - `debug`
